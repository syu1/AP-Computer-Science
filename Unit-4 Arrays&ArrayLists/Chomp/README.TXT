Target: Apply 2D arrays to the engine of a game.

Solo Programming Activity

 

* Sync the Chomp folder from GitHub.
* Implement the methods in the CharMatrix.java file based on the comments.
* Play the game!
 

Extension

* figure out which player, if they know the strategy, will always win (finite game theory)
* modify game to support two human players or two computer players
* modify game for a different board size (note: strategy will no longer work)
* create strategy for game of a different board size


The Chomp case study was devloped by Litvin and Litvin: <http://www.skylit.com/javamethods/index.html>
